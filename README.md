# Qt版本：5.14.1
水产养殖主题的上位机界面程序

心知天气api目前获取不了
数据库文件位置需要自己修改

功能：
1. 用户认证与管理：开发完整登录注册体系，支持新用户注册、密码修改及“记住我”自动填充功能，通过密码框显隐切换图标增强输入安全性，基于SQLite3实现用户数据安全存储与高效校验。
3. 主界面开发：集成心知天气API获取实时天气、精准显示时间日期，支持个性化用户头像展示，移植第三方UI库。
4. 环境监测模块：实现水温、盐度、溶氧量等6项核心环境参数实时刷新，动态展示传感器设备工作状态，支持远程启停环境监测与控制设备。
5. 视频监控模块：基于V4L2驱动开发摄像头控制功能，支持设备启停、实时视频流显示、一键抓拍及图片自动存储。
6. 网络通信模块：实现TCP服务器/客户端双模式，支持指定端口监听、客户端连接及双向数据收发，为系统远程监控与管理提供灵活通信能力； 
7. 商品交易模块：设计水产品表格化展示、购物车增删、实时总价计算及订单结算功能，搭建便捷电子交易平台，简化养殖产品购销流程。
